In order to compile, open the 'MxtUpdater.vcxproj' file with Microsoft Visual Studio 2015,
and make sure the following settings are correct:

main tools bar -> Solution configuration: Win8Beta Release
main tools bar -> Solution Platform: Win32 
Project->Properties->Config Properties->General->Platform Toolset: Visual Studio 2015 (v140)
Project->Properties->Config Properties->C/C++>Warning Level: Level2(/W2)
